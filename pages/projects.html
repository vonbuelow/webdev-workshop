Try it: Add another project card

Copy one of the existing <article class="project-card"> blocks above (including the title, paragraph, and tag). Paste it inside the projects-grid section and change the title and description to a project of your own.

Try it: change the style the project cards

Open styles/projects.css and change the background color of .project-card. Save and refresh to see your change. Try a color like #e8f4f8 or #fff3e0.
