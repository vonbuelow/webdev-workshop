Reference & examples

Use this page as a reference later. No exercises here—just examples you can copy or tweak.

Padding examples

Each box below uses a different padding value. Open styles.css and search for ref-padding to change the values and see what happens.

.ref-padding-tight
padding: 0.5em; — same on all sides
.ref-padding-medium
padding: 1em; — a bit more room
.ref-padding-loose
padding: 2em; — lots of space
.ref-padding-asymmetric
padding: 1em 2em; — vertical, then horizontal
.ref-padding-four
padding: 0.5em 1em 1.5em 2em; — top, right, bottom, left

Margin examples

Each box below uses a different margin value. Margin adds space outside the element (unlike padding, which adds space inside). Open styles.css and search for ref-margin to change the values.

.ref-margin-tight
margin: 0.5em;
.ref-margin-medium
margin: 1em;
.ref-margin-loose
margin: 2em;
.ref-margin-asymmetric
margin: 1em 2em; — vertical, horizontal
.ref-margin-four
margin: 0.5em 1em 1.5em 2em; — top, right, bottom, left
