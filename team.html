Try it: Add two more team members

Duplicate the <div class="team-card"> block above (the one with Alex). Paste it twice inside team-list, then change the name, role, and short bio for two new team members.

You’ll use: the same structure—<div class="team-card">, <h3>, <p class="role">, and a plain <p> for the bio.

Try it: Style the team cards

In styles.css (or a new team.css if you prefer), add a class for the team cards and give them a background color, padding, and maybe a border or border-radius so they stand out from the page.

You’ll use: .team-card { ... } with properties like background-color, padding, border, border-radius.

Try it: Add a “Say hi” button with JavaScript

Add a button to this page, e.g. <button id="say-hi-btn">Say hi</button>. In scripts.js, write code that runs when the button is clicked: show an alert('Hi!') or use console.log('Hi!') and open the browser’s developer console (F12) to see it.

You’ll use: document.getElementById('say-hi-btn') and .addEventListener('click', function () { alert('Hi!'); });.
